







Match h for all TARGET elements:       [mehw h="TARGET"]
Match w for all TARGET elements:       [mehw w="TARGET"]
Match h and w for all TARGET elements: [mehw wh="TARGET"]

Separate multiple targets with a pipe "|".

Responsive support (i.e. if elements drop below each other on smaller screens):

[mehw h="TARGET" responsive="true"] - (default is false)


Use in templates: <?php echo do_shortcode('[mehw h="TARGET"]'); ?>
